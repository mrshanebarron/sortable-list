# Laravel Design Sortable List

Drag and drop sortable list component for Laravel. Supports Livewire, Blade, and Vue 3.

## Installation

```bash
composer require mrshanebarron/sortable-list
```

## Usage

### Livewire Component

```blade
<livewire:ld-sortable-list />
```

### Blade Component

```blade
<x-ld-sortable-list />
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag=ld-sortable-list-config
```

## Customization

### Publishing Views

```bash
php artisan vendor:publish --tag=ld-sortable-list-views
```
